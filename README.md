# Analisador de Nome em Python

Este é um projeto simples desenvolvido durante meus estudos de Python.

O programa solicita o nome do usuário e apresenta algumas informações sobre ele, como:

- Nome em maiúsculo e minúsculo
- Quantidade de letras
- Verificação de espaços
- Nome invertido
- Primeira e última letra

## Tecnologias usadas

- Python

## Objetivo

Praticar conceitos básicos de programação como:

- entrada de dados
- manipulação de strings
- estruturas condicionais
